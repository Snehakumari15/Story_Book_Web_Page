# Story World

Story World is a responsive digital storybook created using **HTML** and **CSS**. It contains 17 inspiring moral stories with beautiful images and an interactive book-like navigation system.

##  Features

-  17 Moral Stories
-  Attractive and Responsive UI
-  Story Images
-  Book-style Navigation
-  Mobile Friendly Design
-  Pure HTML & CSS (No JavaScript)

##  Technologies Used

- HTML5
- CSS3
